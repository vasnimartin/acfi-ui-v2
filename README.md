# ACFI Church UI v2

This contains the version 2 user interface for the ACFI Church website, built with Angular and SCSS.

## Features
-   **Modern Design:** Dark Navy & Gold color scheme with `Cinzel` and `Lato` typography.
-   **Responsive:** Fully responsive layout for Desktop, Tablet, and Mobile.
-   **Components:** Modular components for Hero, Services, Events, Gallery, and more.
-   **Global Styles:** Scalable SCSS architecture with CSS variables.

## Tech Stack
-   **Framework:** Angular 19
-   **Styling:** SCSS
-   **Icons:** FontAwesome

## Development

To start a local development server, run:

```bash
ng serve
```

Navigate to `http://localhost:4200/`.

## Build

To build the project run:

```bash
ng build
```
